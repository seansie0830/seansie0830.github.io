Several ways to convert decimal number to binary in c++

Oct 04, 2023
2 minute read

built-in function

#include<bits/stdc++.h>
using namespace std;
string tobinary(int n){
    char s[128];
    string ss;
    itoa(n,s,2);
    ss=s;
    return ss;
}
int main(){
    int n;
    while(cin >>n){cout<<tobinary(n)<<endl;}
    return 0;
}

In this approach, I use built-in function itoa to convert integer to number of any base in string format.

Also, its return type is char style string ,so be sure to convert to string to make it better to use(You can simply use = to assign char style string to std::string)

prototype

char * itoa ( int value, char * str, int base );

recursion (Most concise)

#include<bits/stdc++.h>
using namespace std;
string tobinary(int n){
    if(n==0) return "";
    return tobinary(n>>1)+((n&1)?"1":"0");
}
int main(){
    int n;
    while(cin >>n){cout<<tobinary(n)<<endl;}
    return 0;
}

concept of this code

In fact , recursion is a tricky concept in my opinion. Therefore, I would use some case to explain this recursive code.

For instance, if n=20 ,when the functiontobinary is called , it would return tobinary(10)+"0" (P.S >> is a shift operator.

And then, it would return "tobinary(5)+"0"+"0" , then "tobinary(2)+"1"+"0"+"0" , then tobinary(1)+"0"+"1"+"0"+"0" . and lastly, it would be ""(due to tobinary(0))“1”+“0”+“1”+“0”+“0”, and final result would be "10100".

things to explain

To put it simplify, n»1 is like n/2 (ignore remainder) but faster that n/2 because it is bit operation.

Also, n&1 is applied for knowing if n is an odd number. If n is an odd number , the result of n&1 would be true .(See picture below)

((n&1)?"1":"0") is one-line style “if” to make its expression more concise.
