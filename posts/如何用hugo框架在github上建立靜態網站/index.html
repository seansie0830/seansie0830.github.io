如何用hugo框架在github上建立靜態網站

Apr 26, 2024

  1. https://winget.run/ 搜尋hugo.

  2. 點擊hugo（不是extend的那一個），複製指令 winget install -e --id Hugo.Hugo， 貼到終端機

  3. 在終端機中輸入hugo驗證安裝，如果成功會出現

PS C:\Users\seans> hugo
Total in 2 ms
Error: Unable to locate config file or config directory. Perhaps you need to create a new site.
Run `hugo help new` for details.
PS C:\Users\seans>

4.建立一個 工作用 資料夾，並在裏面打開終端機

可以用以下方式，在檔案總管上面輸入cmd按enter。

  1. 找個喜歡的主題吧 https://themes.gohugo.io/

https://gohugo.io/getting-started/quick-start/

推薦這個

https://github.com/nanxiaobei/hugo-paper

6.輸入 hugo new site [YOUR-BLOG-NAME] 在終端機中

result would be

PS C:\Users\seans\codewhitelist\newnew> hugo new  site myblog
Congratulations! Your new Hugo site was created in C:\Users\seans\codewhitelist\newnew\myblog.

Just a few more steps...

1. Change the current directory to C:\Users\seans\codewhitelist\newnew\myblog.
2. Create or install a theme:
   - Create a new theme with the command "hugo new theme <THEMENAME>"
   - Or, install a theme from https://themes.gohugo.io/
3. Edit hugo.toml, setting the "theme" property to the theme name.
4. Create new content with the command "hugo new content <SECTIONNAME>\<FILENAME>.<FORMAT>".
5. Start the embedded web server with the command "hugo server --buildDrafts".

See documentation at https://gohugo.io/.
PS C:\Users\seans\codewhitelist\newnew>

P.S.可以直接照裏面做

7.輸入 cd .\ [YOUR-BLOG-NAME]\themes\

  1. 找到主題後，把它clone下來，或zip檔案下載並解壓縮到[工作資料夾]\ [YOUR-BLOG-NAME]\themes

P.S 沒有安裝git不可以用clone 若沒有要自己安裝 winget install -e --id Git.Git

e.g

result would be

PS C:\Users\seans\codewhitelist\newnew\myblog\themes> git clone https://github.com/CaiJimmy/hugo-theme-stack.git

9.打開 ”工作資料夾“\hugo.toml，依如下指示編輯

先加上theme ='YOUR-THEME-FOLDER-NAME' baseURL 改成 localhost

baseURL = 'https://example.org/'
languageCode = 'en-us'
title = 'My New Hugo Site'
theme = 'hugo-theme-stack'

加上theme這一行 ，並且hugo-theme-stack改成你剛剛下載的主題的資料夾名字 theme沒有S!!

10.在 工作資料夾[YOUR-BLOG-NAME] 中打開終端機(參考第5點)

在終端機輸入 hugo new content posts/[your-post-name].md

result


PS C:\Users\seans\codewhitelist\newnew\myblog> hugo new content posts/my-first-post.md
Content "C:\\Users\\seans\\codewhitelist\\newnew\\myblog\\content\\posts\\my-first-post.md" created

11.打開 工作資料夾\content\posts\YOUR-POST-NAME.md

會看到

+++
title = 'My First Post'
date = 2024-04-26T20:07:10+08:00
draft = true
+++

先把draft=true 改成 false

然後下面可以開始寫你的內容(markdown 格式

15.確認沒有問題後，登入github

按右上角的+號，按new repository

接著 *Repository name填入 YOUR-GITHUB-ID.github.io ，按下create repository

15修改hugo.toml(在工作資料夾\YOUR-BLOG-NAME)的baseURL，

格式:https://[你的githubID].github.io

baseURL = 'https://seansie0830.github.io' // 改成https://YOUR-GITHUB-ID.github.io
languageCode = 'en-us'
title = 'My New Hugo Site'
theme = 'hugo-theme-stack' // 加上這一行 hugo-theme-stack改成你剛剛下載的主題的資料夾名字

16

回到工作資料夾，先在終端機輸入hugo ，重新產生正式的網頁

打開裡面的public資料夾(hugo產生的網站路徑)，並打開終端機(見第5點)

輸入(請先安裝git見第8點)

git init
git add .
git commit -m "first commit"

17 最後打開剛剛第14點建立的repository
